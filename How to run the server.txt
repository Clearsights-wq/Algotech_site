Download the zip archive here:
https://limewire.com/?referrer=pq7i8xx7p2
Extract the contents of the archive on some directory.
Go to root project directory
Make sure python is installed.
Open cmd in top directory and type :
python -m http.server 8000
Then type localhost:8000 on some browser.
Finally enter on ZoomOnScroll.html and start 
scrolling slowly to show the website.
